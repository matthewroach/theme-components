# Link

Link can be used to provide an anchor link for either an internal or external URL.

## Usage

```jsx

import { Link } from '@matthewroach/theme-components';

const Feature = () => (
	<Link href="https://github.com/matthewroach">github.com/matthewroach</Link>
);

```
